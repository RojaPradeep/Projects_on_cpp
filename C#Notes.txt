Explain SOLID principles with one real example (from Web API if possible).
Single Repository Principle:Each class should responsible to perform single task meaning it would needs modification for only single task
Open closed principle: Class should provide closed for modification of behaviour but open for extension the functionality.
Liskov substitution priciple:derived class is should be able to replace the functionality of base class.
interface segregation principle:Seperate interface need to create for each task so that no need to force all the classes to implement all the behaviour.
Depencency injection principle:High level models no need to depend on the low level models  both depend on abstractions

What is a delegate?
Delegates is type safe function pointer  which holds the references to one or more methods when the signature matches and allows them to invoke.

namespace ConsoleApp1
{
   
    internal class Program
    {
        static void Main(string[] args)
        {
            NotificationMethods notification = new NotificationMethods();
            Notify notify = notification.SMSNotification;
            notify += notification.EmailNotification;
            notify?.Invoke("message is sent");
            Console.ReadKey();
        }
    }
    class NotificationMethods
    {
        public void  SMSNotification(string message)
        {
            Console.WriteLine("SMS Notification is send");
        }
        public void EmailNotification(string message)
        {
            Console.WriteLine("Email Notification is send");
        }
    }
